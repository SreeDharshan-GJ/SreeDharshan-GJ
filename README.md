### Smart Traffic Control using Reinforcement Learning
Multi-intersection traffic signal optimization using deep Q-networks. Trained agents in a simulated environment to minimize average vehicle wait times and improve throughput without centralized coordination.

### Image Caption Generation using CNN-LSTM
End-to-end image captioning model using a pretrained CNN encoder (ResNet) with an LSTM decoder and attention mechanism. Trained and evaluated on the MS-COCO dataset with BLEU score benchmarking.

[![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)](https://github.com/SreeDharshan-GJ-Research/Image-Caption-Generator-using-CNN-LSTM)
